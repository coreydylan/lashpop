# OpenGraph Image Requirements

## Current Status
The site references `/images/og-image.jpg` in the metadata but the image may not exist.

## Required Specifications

### Dimensions
- Width: 1200px
- Height: 630px (1.91:1 aspect ratio)

### Content Recommendations
- Include LashPop Studios logo
- Show beautiful lash work or studio interior
- Include tagline: "Naturally Effortless Lashes + Beauty"
- Include location: "Oceanside, CA"
- Use brand colors (dusty rose, cream, sage)

### Technical Requirements
- Format: JPEG or PNG
- File size: Under 300KB for fast loading
- No important content in outer 10% margins (safe area)

### Placement
Save to: `/public/images/og-image.jpg`

### Also Create
- Twitter card image at `/public/images/twitter-card.jpg` (same dimensions work)

## Tools for Creation
- Canva (free templates available)
- Figma
- Adobe Express
