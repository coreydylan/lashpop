# Vagaro API Reference Documentation

Complete reference guide for integrating with the Vagaro API for beauty, fitness, and wellness business operations.

## Overview

Vagaro provides a comprehensive API for managing appointments, customers, employees, and business operations across multiple locations. This reference covers both REST API endpoints and webhook integrations.

## Key Features

- **Multi-location Support** - Manage employees and services across multiple business locations
- **Real-time Updates** - Webhook notifications for appointments, customers, and transactions
- **Comprehensive Booking** - Full appointment lifecycle management with availability search
- **Customer Relationship** - Complete customer profile and interaction tracking
- **Access Control** - Role-based permissions and access levels
